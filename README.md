# Bank Project in C

This is a simple banking system project written in C.  
The main goal is to practice core programming concepts such as structs, pointers, dynamic memory allocation, and modular code organization.

## Features

- User registration (username and password)
- Dynamic allocation of user data
- Array of users managed with pointers
- Basic functions for printing and freeing allocated memory

## Compilation

To compile the project, run:

```bash
make
